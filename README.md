# CyberConnect SOC Fellowship Repo

## Overview

This repository documents my weekly progress in the **CyberConnect SOC Fellowship Program**.

Each week we are assigned Capture The Flag (CTF) challenges to strengthen practical cybersecurity skills such as:

* Linux fundamentals
* Threat investigation
* Security analysis
* Problem solving
* Security tools usage

I will upload detailed **write-ups, commands, screenshots, and solutions** for every challenge I solve during the fellowship.

# CyberConnect SOC Fellowship Repo

## Overview

This repository documents my weekly progress in the **CyberConnect SOC Fellowship Program**.

Each week we are assigned Capture The Flag (CTF) challenges to strengthen practical cybersecurity skills such as:

* Linux fundamentals
* Threat investigation
* Security analysis
* Problem solving
* Security tools usage

I will upload detailed **write-ups, commands, screenshots, and solutions** for every challenge I solve during the fellowship.

---

## Track

The goal of this track is to develop practical skills used by SOC analysts such as:

* Log analysis
* Threat detection
* Incident investigation
* Security monitoring
* Understanding attacker techniques

## Repository Structure

Each challenge has its own folder containing:

* `README.md` → Explanation of how the challenge was solved
* `solution.png` → Screenshot proof of the solved challenge

Example structure:

Week-1

        ├── README.md
        └── solution.png
